# FoodGuard
An android app that tracks food expiry dates to help individuals reduce food waste. Features include tags, filters, search function, and OCR image to text date recording. 

# Additional Info
This is a project for the OSRF 2026 in collaboration with Korn Swatdipong my ORSF partner. He developed the IOS version while I made the android version using Flutter. 
# How to Run

Install Flutter.

Clone the repository:
git clone https://github.com/Toki-dev465/FoodGuard.git

Navigate to the project folder:
cd FoodGuard

Install dependencies:
flutter pub get

Run the app:
flutter run
